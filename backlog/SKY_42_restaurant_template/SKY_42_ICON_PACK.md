## Icon Usage Guidelines

### Size Scale
- **UI icons:** 24x24px (header, buttons)
- **Category icons:** 20px (tabs), 24px (section headers)
- **Badge icons:** 14-16px (inline with text)

### Color Strategy
- **Default:** `currentColor` (inherits from parent)
- **Category icons:** Primary color (#E63946) when active
- **Badge icons:** Specific colors per type (see badge specs)
- **UI icons:** Text color (#1A1A1A) or white on dark backgrounds

### Accessibility
- All icons paired with text labels (not standalone)
- ARIA labels when icon-only: `aria-label="Add to cart"`
- Screen reader friendly: `<span class="sr-only">Cart</span>`

### Optimization
- **SVG sprite:** Combine all icons into single sprite.svg (reduce HTTP requests)
- **Inline critical:** Inline UI icons (cart, search, close) for instant render
- **Lazy load:** Category icons loaded with sections (not critical path)

### Fallback Strategy
```jsx
// Emoji fallback if SVG fails
<span className="icon" aria-label="Burgers">
  <svg>...</svg>
  <span className="emoji-fallback">üçî</span>
</span>
```

### React Component Example
```tsx
// components/Icon.tsx
export const Icon = ({ name, size = 24, color = "currentColor" }) => {
  const icons = {
    burger: BurgerIcon,
    cart: CartIcon,
    search: SearchIcon,
    // ... map all icons
  };

  const IconComponent = icons[name];
  return <IconComponent width={size} height={size} fill={color} />;
};

// Usage
<Icon name="burger" size={20} color="#E63946" />
```
